# 📅 Day 8 - Pattern Printing in JavaScript

## 🚀 Overview

On Day 8, I practiced different types of pattern printing using JavaScript.
This helped me strengthen my understanding of **loops, nested loops, and logic building**.

---

```
*
* *
* * *
* * * *
* * * * *
```
