📅 Day 6 - Switch Case (JavaScript)
🧠 Topic Covered
Switch Case statement in JavaScript
Handling multiple conditions efficiently
💻 Problem Description

Take a number input from the user (1–7) and print the corresponding day of the week.

⚙️ Approach
Took user input using prompt()
Used a switch statement to match the input value
Printed the respective day for each case
Used a default case to handle invalid inputs
📌 Code Logic (Short Explanation)
If input is 1 → Monday
If input is 2 → Tuesday
...
If input is 7 → Sunday
Otherwise → "Invalid input"
