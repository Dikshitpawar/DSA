# 📘 DSA Day 9 - Arrays Practice in JavaScript

## 🚀 Topics Covered

This day focuses on basic array operations and problem-solving techniques using JavaScript.

---

## 🔢 1. Sum of Array Elements

Calculate the sum of all elements in an array.

### Logic:

* Initialize sum = 0
* Traverse the array
* Add each element to sum

---

## 🔝 2. Maximum Element in Array

Find the largest element in the array.

### Logic:

* Assume first element as max
* Compare with each element
* Update max if a larger value is found

---

## 🥈 3. Second Maximum Element

Find the second largest element.

### Logic:

* Maintain two variables: max and second max
* Update accordingly while traversing

---

## 🔄 4. Reverse an Array

### Method 1 (Using Extra Array):

* Traverse from end and store in new array

### Method 2 (Two Pointer):

* Swap elements from start and end

---

## ⚪⚫ 5. Move All 0s to Left & 1s to Right

### Logic:

* Use pointer `j`
* Swap whenever 0 is found

---

## 🔁 6. Left Rotate Array by 1

### Logic:

* Store first element
* Shift all elements left
* Place first element at end

---

## 🔁 7. Left Rotate Array by K Elements

### Method 1:

* Perform left rotation K times

### Method 2 (Efficient):

* Use modulo
* Create new array using `(i + k) % n`

### Method 3 (Efficient):

* Reverse first k elements
* Reverse remaining elements
* Reverse entire array
