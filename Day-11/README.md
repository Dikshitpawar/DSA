# Day 11 - Sorting Algorithms in JavaScript

Today I practiced and implemented three fundamental sorting algorithms in JavaScript:

## Algorithms Covered

### 1. Bubble Sort
- Repeatedly compares adjacent elements.
- Swaps them if they are in the wrong order.
- Largest element moves to its correct position in each pass.

**Time Complexity:**
- Best Case: O(n)
- Average Case: O(n²)
- Worst Case: O(n²)

---

### 2. Selection Sort
- Finds the smallest element from the unsorted portion.
- Places it at the beginning of the sorted portion.
- Reduces the unsorted array size in each iteration.

**Time Complexity:**
- Best Case: O(n²)
- Average Case: O(n²)
- Worst Case: O(n²)

---

### 3. Insertion Sort
- Builds the sorted array one element at a time.
- Inserts each element into its correct position.
- Efficient for small or nearly sorted datasets.

**Time Complexity:**
- Best Case: O(n)
- Average Case: O(n²)
- Worst Case: O(n²)
