# Day 14 - String Problems in JavaScript 🚀

Today I practiced some important string manipulation problems using JavaScript. These problems helped me improve my understanding of loops, string methods, arrays, and the two-pointer technique.

## Problems Solved

### 1. Print Each Character on a New Line
- Accept a string from the user.
- Print every character on a separate line.
- Concept Used: `for...of` loop.

### 2. Print String in Reverse Order
- Accept a string from the user.
- Print the string in reverse order.
- Concept Used: Reverse traversal using loops.

### 3. Reverse a String Using Two Pointer Algorithm
- Reverse a string by swapping characters from both ends.
- Concept Used: Two Pointer Technique.

### 4. Toggle Case of Each Alphabet
- Convert uppercase letters to lowercase and lowercase letters to uppercase.
- Example:
  - Input: `AcgDfD`
  - Output: `aCGdFd`
- Concept Used: ASCII values and string manipulation.

### 5. Count Words with Given Prefix
- Given an array of words and a prefix, count how many words start with that prefix.
- Example:
  - Input: `["pay","attention","practice","attend"]`, Prefix: `at`
  - Output: `2`
- Concept Used: `startsWith()` method.

### 6. Capitalize First and Last Character of Each Word
- Convert the first and last character of every word to uppercase.
- Example:
  - Input: `Hello bhai Kaise ho a`
  - Output: `HellO BhaI KaisE HO A`
- Concept Used: String slicing and character manipulation.
